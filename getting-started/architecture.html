ExChron Architecture (Simple Overview)

Overview at a glance​

Exchron has two parts that work together:

  • The interactive app where you explore data and build models.
  • The documentation site you’re reading now.

Both are designed to be fast, simple to use, and easy to learn—whether you’re a beginner or an expert.

How the App Is Organized​

Exchron groups your work into two main experiences:

  • Playground Mode: A place to quickly try models, classify data, and see results.
  • Classroom Mode: A guided path that takes you from data to export step by step.

The User Flow​

  1. Start at the home page and choose Playground or Classroom.
  2. In Playground, pick a model, add your data, and view results. If needed, refine and try again.
  3. In Classroom, follow the flow: Data Input → Model Selection → Train & Validate → Test & Export.
  4. Use the built-in assistant anytime if you need help.

What Happens Behind the Scenes (At a High Level)​

  • Your actions in the app (like choosing a model or uploading data) trigger the right tools to run and produce results.
  • Results are shown clearly so you can understand what happened and what to try next.
  • You can export your outcomes to use or share them elsewhere.

Why This Structure​

  • It keeps learning and experimenting separate but close—so you can move between them easily.
  • It helps beginners get step-by-step guidance, while giving advanced users a fast way to iterate.
  • It supports future additions like new models and features without changing how you use the app.

Where to Go Next​

  • Try the Quickstart to walk through the workflows.
  • See the User Guide for deeper help on Playground and Classroom.
  • Visit the demo to try Exchron in your browser.
